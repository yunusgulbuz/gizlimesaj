Ödeme Linki Oluştur
Link ile ödeme alma
Ödeme Linki oluşturma uygulaması ile kendi ödeme linkinizi oluşturabilirsiniz.

API ile üye işyeri, kendi ödeme linkini oluşturup bu link üzerinden ödeme alabilir.

Postman collection’ını indirmek için tıklayınız. Örnek kodları Postman Code sinppet’inde görebilirsiniz.

Başarılı işlem sonrası, işlem sonucu girmiş olduğunuz success URL’e POST edilir.

Bu servis için hash oluşturma: Ortak Ödeme ve API’ler için Hash oluşturma ile aynıdır. Hash methodu için tıklayınız.

Ödeme Linki Oluşturma API Kullanımı
Bu API’nin kullanımında, ilgili parametreler “https://paynkolaytest.nkolayislem.com.tr/Vpos/by-link-create” adresine Body’de form-data olarak POST edilir.
